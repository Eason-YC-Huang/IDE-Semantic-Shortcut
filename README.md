# IDE-Semantic-Shortcut
IDE - Semantic - Shortcuts are shortcuts that have semantic meaning which can help you memorize more efficiently.

- 便捷快捷键 OK

  设置页面：ctrl + ,

  复制：ctrl + c

  粘贴：ctrl + v

  剪切：ctrl + x

  撤销：ctrl + z

  重做：ctrl + shift + z

  复制当前行 duplicate line or selection：ctrl + d

  删除当前行 delete line：ctrl +y

  选中全部 select all ：ctrl + a

  扩大选区 extend selection：ctrl + w  (not in vscode)

  完成当前语句 complete current statement：ctrl + shift + enter (not in vscode)

  关闭页面close tab ： ctrl + alt + w

  回到上一次浏览 navigate back / forward ：ctrl + alt + left/right

  上下移动行 move statement up / down ：alt + up/down

  光标移动到行首/行末 move caret to line start / end ： ctrl + left/right

  光标移动到行首/行末并选中 move caret to line start / end with selection : ctrl + shift + left/right

  新增或者删除光标Add or Remove Caret ： alt + left-click

  拖动新增或删除光标Add Rectangular Selection on Mouse Drag ：  alt + shift + left-click

  左右切换标签页 select previous/next tab → ctrl + alt + shift + left/right

  go to declaration / usage → ctrl + left-click

  go to implementation → ctrl + alt + left-click

  go to type declaration → ctrl + alt + shift + left-click
